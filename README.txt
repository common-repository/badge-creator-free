The easiest way to create event’s badge with WordPress and share on your social networks (Facebook, WhatsApp, twitter, etc…).

== Description ==
This plugin makes it very easy for your website users to create badges for anything ( from Facebook profile picture badge to showing support for social events!). The principle is simple:
1- create a badge
2- Put the shortcode of this badge in any page or article (pretty much everywhere that you wish to put it!)
3- Let you users have fun!

And that's all! Each of your visitors to upload an image of him, save his badge and use it! 
Badge design has never been easier!

== Installation ==

Upload the entire plugin's folder to the /wp-content/plugins/ directory.
Activate the plugin through the ‘Plugins’ menu in WordPress.
